How to apply your changes

Once you've made your changes and clicked Save .sav, the browser downloads a file (default name <name>.edited.sav). To make the game use it, you need to copy that file into the game's save directory — overwriting the original.

1 · Find your Windows username

Press Win+R, type whoami, press Enter. Copy the result (e.g. Alejm) into the box below — it's stored only in this browser.

2 · The full path

This is the exact file you need to replace:

On macOS the path is ~/Library/Application Support/Godot/app_userdata/Fortune Mill/save_game.sav; on Linux it's ~/.local/share/godot/app_userdata/Fortune Mill/save_game.sav.

3 · Steps

  1. Close Fortune Mill. The game overwrites the save when it exits, so it must be closed first.
  2. Back up the original. Rename the current save_game.sav to save_game.sav.bak (just in case).
  3. Copy your edited file into the folder above, named save_game.sav.
  4. Reopen the game. Your changes should be in effect.
💡 Tip — write directly: your browser supports the File System Access API. You can skip the copy step and write the file straight to the game's folder. Click and pick save_game.sav in the file dialog.

The Fortune Mill save format was reverse-engineered by HolzMichel (original guide). This editor is a fan project built on top of that work.
